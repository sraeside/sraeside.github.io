Dog Days

I designed and developed Dog Days, a responsive product landing page for a fictional company specializing in high-quality walk-kits for dogs. This project showcases my ability to create accessible, well-structured web interfaces using semantic HTML5 and CSS3, ensuring seamless navigation and an engaging user experience.

In crafting Dog Days, I focused on delivering a visually engaging and user-friendly experience while ensuring high accessibility standards. Thoughtful CSS design choices and proper selectors were employed to maintain an elegant and consistent aesthetic. The combination of Flexbox and Grid enabled a balanced layout that accommodates various screen sizes, reinforcing the importance of responsive web development.

This project demonstrates my ability to build structured, functional, and interactive web pages while maintaining compliance with modern development standards. It reflects my commitment to intuitive UX design, accessibility, and clean, maintainable code.

the kitchen

I designed and developed a website for The Kitchen, a fictional Italian restaurant. This project showcases my proficiency in HTML5 and CSS, implementing best practices in semantic structure, external styling, and user-friendly navigation.

The website follows a clean, semantic HTML structure, ensuring accessibility and proper document flow. Styling is applied through externally linked CSS, balancing aesthetics and readability. While responsiveness was not a requirement, I incorporated media queries for enhanced usability across different screen sizes, demonstrating attention to modern web design principles.

The Kitchen website reflects my ability to create structured, visually appealing, and user-friendly web experiences. This project solidified my understanding of HTML and CSS, emphasizing thoughtful design, navigation, and multimedia integration.
